Have ID columns with leading zeros (like "007")?
?
CSV files preserve leading zeros when the zeros are already present in the CSV. For Excel, format the ID column as Text before entering values — otherwise Excel itself strips the leading zero before we ever see the file (and this also affects a CSV exported from that same Excel file). For JSON, make sure ID values are quoted strings ("customer_id": "007"), not bare numbers. Once a leading zero is lost in the source file, it can't be recovered after upload.
