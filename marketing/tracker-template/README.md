# Archimedes Marketing Tracker Template

This is a free, Google Sheets-style tracker template. Nothing here connects to live accounts or paid tools.

## How To Use

Create a new Google Sheet, then import each CSV as its own tab:

- `weekly_scorecard.csv`
- `inquiry_log.csv`
- `campaign_tracker.csv`
- `content_tracker.csv`
- `utm_builder.csv`
- `future_calendly_tracker.csv`
- `internal_traffic_exclusions.csv`

## Important Tracking Rule

Do not count visits from Archimedes, Ellie, or internal testing as marketing traffic.

Use `internal_traffic_exclusions.csv` to record known internal devices, browsers, IP ranges, test notes, and any dates where traffic was inflated by testing.

Until analytics is connected later, internal visits should be filtered manually when reviewing traffic.

## Current Funnel

The current funnel is designed for a pre-customer stage:

1. Website visit
2. Engaged visit
3. Contact/email section interest
4. Email inquiry
5. Qualified inquiry
6. Follow-up sent
7. Outcome

Calendly is intentionally future-only for now.
