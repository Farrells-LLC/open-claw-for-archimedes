# OpenClaw Workspace Graph

This workspace is an Obsidian-ready knowledge graph for OpenClaw memory.

Start here:

- [[MEMORY]] - curated long-term memory, used only in private/main sessions
- [[HEARTBEAT]] - proactive checks and maintenance prompts
- [[AGENTS]] - workspace operating rules
- [[SOUL]] - Ellie identity and working style
- [[USER]] - Mr. Farrell context
- [[TOOLS]] - local tool notes
- [[second-brain/README|Second Brain]] - structured knowledge base
- [[directives/README|Directives]] - repeatable workflows and SOPs

Daily raw logs live in [[memory/]]. Structured summaries and durable knowledge live in [[second-brain/]].

## Memory Flow

1. Capture raw context in `memory/YYYY-MM-DD.md`.
2. Promote stable lessons and decisions into `MEMORY.md`.
3. Create structured notes in `second-brain/` for concepts, projects, documents, and summaries.
4. Use `directives/` for repeatable workflows the agent should follow later.

## Obsidian Setup

Open this folder as an Obsidian vault. The included `.obsidian/` settings enable core graph, templates, daily notes, backlinks, and tags. Recommended community plugins:

- Dataview
- Templater
